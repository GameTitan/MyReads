## My Reads

My Reads is an application used to organize different bookd onto shelves. You can also add books to shelves and then move them around. This Application was made as a part of the React JS Course on Udacity.com

## Installation
 
To get this up and running run these commands in the project folder

```
	npm install
	npm start
```
